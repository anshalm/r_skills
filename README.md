# r_skills
Hi! This is a large-scale version of my regex project, but now 
with more complex querying, datasets, and visualization. Thanks 
for the watch!

## What This Project Covers
- Regex & stringi — string extraction and text cleaning
- dplyr — filter, select, mutate, rename, relocate, 
  group_by, summarise, arrange
- plotly — interactive bar chart visualization
- Data import — read.csv for multiple datasets
- Data cleaning — removing unrated values, 
  type conversion, unlisting

## Datasets
- Username CSV — regex extraction of structured string data
- Ramen Ratings Dataset from Kaggle
  https://www.kaggle.com/datasets/residentmario/ramen-ratings
